### JS-Video

This is a web that uses the package [@mer_ren22/jsmediaplayer](https://www.npmjs.com/package/@mer_ren22/jsmediaplayer) it loads it and makes use of all the plugins which include:

- ads
- auto play
- auto stop
- Mute/Unmute
